Simple To-Do List

This is a beginner-friendly **To-Do List** app made with **HTML, CSS, and JavaScript**. A to-do list is just a list of things you have to-do. That means basically anything and everything can be on your to-do list

Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  

How it works
Type a task in the input box and click **"Add"**.
